# Лабораторная работа №5

## Цель

Решение задачи условной оптимизации методом барьерных функций с внутренними итерациями на базе существующего непрерывного метода Розенброка из ЛР2.

## Внешний цикл

В ЛР5 внешний барьерный цикл работает по общей параметрической схеме:

```text
mu_1 = mu0
mu_{k+1} = beta * mu_k, 0 < beta < 1
```

Остановка внешних итераций:

```text
mu_k * M(x_{mu_k}) < epsilon_outer
```

где `M(x) = Σ 1 / (-g_i(x))` - общая положительная метрика близости к границе допустимой области.
Для reciprocal barrier эта метрика совпадает по смыслу с методической схемой.
Для log barrier сама барьерная функция в внутренней цели остаётся `-Σ ln(-g_i(x))`, а внешний стоп-критерий использует ту же общую метрику `M(x)`, чтобы не зависеть от знака лог-барьера.
В результате сохраняется последняя допустимая внешняя итерация; если следующий шаг нарушает допустимость или численный контракт, расчёт переходит в `warning`/`error` без подмены invalid-точки итогом.

## Запуск GUI

```bash
cd optimmethods
source .venv/bin/activate
python -m lr5
```

В GUI слева находятся `Основные` параметры, а блок `Дополнительные` скрыт по умолчанию.
Справа - вкладки `Отчёт` и `График`.
До расчёта показывается empty state с подсказкой.
После запуска в отчёте появляется карточка итога, таблица внешних итераций и вторичная секция подробностей внутренних итераций.
Карточка итога теперь разделяет `Готово`, `Расчёт остановлен с предупреждением` и `Ошибка`, а в таблице внешних итераций показываются только валидные допустимые шаги.

## Запуск отчётного режима

Если нужен headless-режим с CSV/PNG/TXT, используй:

```bash
python -m lr5.main
```

## Постановка по умолчанию

По умолчанию запускается задача:

```text
F(x1, x2) = (x1 - 5)^2 + (x2 - 3)^2
g1(x) = -x1 + 2x2 - 4 <= 0
g2(x) = x1 + x2 - 3 <= 0
```

Стартовая точка:

```text
(0, 0)
```

Параметры внешнего цикла по умолчанию:

```text
mu0 = 10
beta = 0.1
epsilon_outer = 1e-3
max_outer_iterations = 20
```

## Артефакты

В GUI есть кнопка `Сохранить результаты`. Она создаёт каталог `report/lr5_runs/<timestamp>_<trace_id>/` со следующими файлами:

- `summary.csv` — таблица внешних итераций;
- `summary.txt` — краткий итог со статусом и последней допустимой точкой;
- `contour.png` — линии уровня, ограничения, допустимая область и траектории;
- `outer_XX/inner_iterations.csv` — история внутренних итераций;
- `outer_XX/trajectory.csv` — траектория внутренних итераций.
