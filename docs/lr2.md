# ЛР2 — Метод Розенброка (непрерывный шаг)

## Запуск

```bash
python3 -m pip install -r lr2/requirements.txt
python3 -m lr2
```

## Что делает GUI

- Принимает полином двух переменных в виде матрицы коэффициентов `c[i][j]`, где
  `f(x1, x2) = sum(c[i][j] * x1^i * x2^j)`.
- Матрица коэффициентов в UI фиксирована: `5x5` (степени `x1^0..x1^4` и `x2^0..x2^4`).
- Поддерживает пресеты из задания:
  - `F1(x) = (3x1^2 - x2)^2 + (2x1 - 3x2)^2`;
  - `F2(x) = 9x1^2 + 16x2^2 - 90x1 - 128x2`.
- Принимает список точностей `epsilon` через набор отдельных полей `ε` с кнопками `+`/`−`.
  По умолчанию отображается одно поле `ε`.
  Прокрутка длинного списка `ε` выполняется внутри этой строки, без прокрутки всего окна.
- Принимает список стартовых точек через набор пар полей `(x1, x2)` с кнопками `+`/`−`.
  Прокрутка длинного списка стартовых точек также ограничена только этой строкой.
- Для каждой пары `(epsilon, start_point)` запускает метод Розенброка
  с непрерывным шагом.

## Размерность

- Ядро solver реализовано для `N`-мерного случая.
- Текущий GUI по-прежнему ограничен 2D-вводом (`x1`, `x2`) и работает как адаптер к N-D solver.

## Формат итерационной таблицы

Вывод соответствует требованию задания:

- `K`, `x_k`, `F(x_k)`;
- `j`, `d_j`, `y_j`, `f(y_j)`, `lambda_j`, `y_{j+1}`, `f(y_{j+1})`.

## Графики

Графики вынесены в отдельную вкладку `Графики` и отображаются в увеличенной области.
Для выбранного запуска доступен переключатель режима:

- `Контуры 2D` — линии уровня функции + траектория с нумерацией шагов;
- `3D поверхность` — поверхность функции + та же траектория.

## Примечание по коэффициентам

- Строка `i` в таблице — степень `x1^i`.
- Столбец `j` — степень `x2^j`.
- Пустая ячейка трактуется как `0`.
- Для плоской функции (например, когда все `c[i][j] = 0`) поиск по направлению
  фиксирует `lambda = 0` и метод корректно завершает расчет за одну итерацию
  без разлета траектории.

## Migration note (breaking change)

- Удален блок выбора степеней и кнопка обновления таблицы коэффициентов.
- Новый контракт UI: коэффициенты всегда задаются в фиксированной таблице `c[0..4][0..4]`.
- Для функций меньшей степени оставляйте старшие коэффициенты равными `0`.
- Контракт solver изменен: метод Розенброка принимает произвольную целевую функцию `objective(vector)` и `N`-мерную стартовую точку.
- Для вырожденных случаев добавлены явные критерии остановки по стагнации значения функции и устойчивый fallback перестройки направлений.
