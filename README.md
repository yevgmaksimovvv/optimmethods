# Методы оптимизации — лабораторные работы

Этот репозиторий содержит материалы и реализации лабораторных работ по дисциплине «Методы оптимизации».

## Структура

- `lr1/` — исходный код первой лабораторной работы.
- `lr2/` — исходный код второй лабораторной работы.
- `lr3/` — исходный код третьей лабораторной работы.
- `optim_core/` — общие контракты и утилиты, переиспользуемые между лабораторными.
- `docs/` — документация по лабораторным работам.
- `tasks/` — постановки и изображения заданий.
- `Отчет/` — отчётные документы.

## Документация по лабораторным

- [ЛР1](docs/lr1.md)
- [ЛР2](docs/lr2.md)
- [ЛР3](docs/lr3.md)

## Режимы запуска

- Отдельные лабораторные:
  - `python3 -m lr1`
  - `python3 -m lr2`
  - `python3 -m lr3`
- Единый статический shell:
  - `python3 -m optim_core.labs`

## Установка и проверка

```bash
cd optimmethods
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -e .[test]
python3 -m pytest -q
```
